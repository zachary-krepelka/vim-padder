# padder.vim

Vim Padder is a plugin for padding lines with characters up to a fixed width.
Full details are provided in the documentation.  Use your favorite plugin
manager for installation.

```text
BEFORE:

     ......       ...... $
   ..........   .......... $
 ............. ............. $
............................. $
............................. $
............................. $
 ........................... $
   ....................... $
     ................... $
       ............... $
         ........... $
           ....... $
             ... $
              . $

AFTER:

     ......       ......      $
   ..........   ..........    $
 ............. .............  $
............................. $
............................. $
............................. $
 ...........................  $
   .......................    $
     ...................      $
       ...............        $
         ...........          $
           .......            $
             ...              $
              .               $
```
